# 01 데이터 타입의 종류

## 기본형(원시형) Primitive type

- number 숫자
- string 문자열
- boolean 불리언
- null
- undefined
- symbol (added ES6)

### 특징

- 할당이나 연산 시 값이 담긴 주솟값을 바로 복제
- 불변성(immutability) 띈다

## 참조형 Reference type

- object 객체
- Date 날짜
- Array 배열
- Function 함수
- RegExp 정규식 표현
- Map, WeakMap (added ES6)
- Set, WeakSet (added ES6)

### 특징

- 할당이나 연산 시 값이 담긴 주솟값들로 이루어진 묶음을 가리키는 주솟값을 복제.
