## 클로저 란

```
어떤 함수에서 선언한 변수를 참조하는 내부함수를 외부로 전달할 경우,
함수의 실행 컨텍스트가 종료된 후에도
해당 변수가 사라지지 않는 현상
```

<br>

<P>내부함수를 외부로 전달하는 방법에는</p>

```
함수를 return 하는 경우뿐 아니라
콜백으로 전달하는 경우도 포함
```

<br>
<p>

클로저는 그 본질이 `메모리를 계속 차지하느 개념`이다</p>

```
더는 사용하지 않게 된 클로저에 대해서는
메모리르 차지하지 않도록 관리해줘야한다.
```
